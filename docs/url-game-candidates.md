# URL候補の実装管理表

参考URL: https://www.spacemarket.com/magazine/know-how/party/drinking-party-game/

スペースマーケットマガジンの記事で紹介されている飲み会ゲーム候補を、静的ページで遊べる形に置き換えながら順番に実装するための管理表です。

市販ボードゲーム、外部アプリ、商標のあるゲームは完全再現せず、飲み会用アプリ内で安全に進行できる汎用ルールとして再構成します。

## 実装方針

- 記事の並び順を基本に進める
- すでにトップゲームや定番ゲームパックで遊べるものは「既存対応」とする
- 専用画面にした方が進行しやすいものからトップゲーム化する
- 各ゲームは、詳しい説明、進行ルール、判定方法、300問相当の出題プールを持たせる
- 静的版ではデータベースを使わず、参加者や進行状態はブラウザ内に保存する
- AWS版への移行時に、部屋、参加者、投票、回答ログをサーバー側へ移せるよう、ゲームごとの状態を分けておく
