# 飲み会用アプリ 要件定義

## 1. アプリ全体

### 目的

飲み会、懇親会、チームビルディング、初対面の交流会で、参加者がすぐ遊べるミニゲーム集を提供する。

### 想定利用シーン

- 会社の懇親会
- 友人同士の飲み会
- オフ会
- 新歓、歓迎会、送別会
- 少人数から中規模の交流イベント

### 想定ユーザー

- 幹事
- 参加者
- 場を盛り上げたい人
- 初対面同士の会話を作りたい人

### 共通要件

- トップ画面にゲーム一覧を表示する
- 各ゲームの入口には、ゲーム名、短い説明、推奨人数、所要時間を表示する
- スマートフォンで操作しやすい画面にする
- ルール説明を各ゲーム開始前に表示する
- トップ画面とゲーム中の画面から、全ゲームの保存状態を初期化できる
- 全体初期化後はトップ画面へ戻る
- 飲み会で使いやすいよう、1ゲームあたり5分から15分程度で遊べる設計にする
- ネガティブないじり、個人攻撃、過度な暴露につながるお題は避ける
- ゲームごとに「やさしい」「普通」「盛り上がる」などの雰囲気を選べる余地を残す

### トップ画面要件

- アプリ名を表示する
- ゲーム一覧をカード形式で表示する
- 各カードからゲーム詳細または開始画面に移動できる
- 初期表示では以下のゲームを並べる
  - 山手線ゲーム
  - ジョハリの窓
  - ウミガメのスープ
  - ワードウルフ
  - NGワードゲーム
  - 二択トーク
  - 第一印象ランキング
  - 定番ゲームパック
  - 匿名質問箱

## URL候補: 山手線ゲーム

### 概要

スペースマーケットマガジンの飲み会ゲーム記事で最初に紹介されている「山手線ゲーム」を、1台共有モードで進行できる専用ゲームとして提供する。

### 目的

- 初対面でも説明が短く、すぐ始められる
- 飲み会の序盤から中盤で場を温める
- 共通体験や会場に合わせたお題で会話を広げる

### 推奨人数

2人から20人程度。

### 所要時間

3分から10分。

### 機能要件

- 参加者名を登録できる
- 通常300問、大人向け30問のお題プールを持つ
- Hな話題をトグルスイッチでON/OFFできる
- Hな話題OFFでは大人向けカテゴリを表示せず、出題にも含めない
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使うお題数を選べる
- カテゴリでお題を絞り込める
- 1人の持ち時間を3秒、5秒、10秒から選べる
- 出た答えを記録できる
- 同じ答えを検知できる
- パスとアウトを記録できる
- お題ごとの例を表示できる
- 終了時にアウト回数を表示できる

### 注意事項

- 判定でもめそうな場合は司会判断でテンポよく進める
- 内輪ネタは、全員が分かる範囲にする
- アウトを責めすぎず、言い間違いや詰まりを笑って次へ進める

## URL候補: 全ゲーム個別入口

### 概要

スペースマーケットマガジンの飲み会ゲーム記事にある候補を、トップ画面から個別に開始できるようにする。外部アプリや市販ゲーム系は完全再現せず、静的ページで遊べる汎用版として提供する。

### 対象

マジョリティゲーム、2つの真実と1つの嘘、カウントアップゲーム、逆さ言葉ゲーム、曲名連想クイズ、お絵描きクイズ、ドキドキはずれカード、スマホ早打ちゲーム、記憶だけでロゴを書く、価値観メーター、ひとこと演技ゲーム、飲み会すごろく、シンプル陣取り、変な一言カルタ、エモヒント連想、資源交渉トーク、人生イベントすごろく、腕相撲トーナメント、安全はずれ抽選、人物当てヒントクイズ、大人数マジョリティ、鼻歌イントロドン、外来語禁止ゲーム。

### 機能要件

- トップ画面にURL候補の個別カードを表示できる
- トップ画面で「全部」「URL候補」「会話」「反射」「運」「描く」「ボード風」「大人数」に絞り込める
- 各URL候補ゲームは通常300問、大人向け30問を持つ
- 各URL候補ゲームはHな話題をON/OFFできる
- 各URL候補ゲームは5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べる
- 各URL候補ゲームは参加者、進行、履歴をブラウザ内に保存できる
- 描画、すごろく、陣取り、トーナメント、抽選などは専用の簡易操作を持つ

### 注意事項

- 音源、画像、外部アプリ連携、専用ボード部品は使わない
- 飲酒強要、危険な罰、身体的に危ない操作は入れない
- 大人向けONでも露骨すぎる話題や答えにくい話題はスキップできる
- AWS多人数版へ移行しやすいよう、ゲーム状態はJSON化しやすい形にする

## 2. ジョハリの窓

### 概要

自分が思う自分の特徴と、他人から見た自分の特徴を比較し、自己理解と相互理解を深めるゲーム。

### 目的

- 初対面同士の会話を作る
- チーム内の相互理解を促す
- 自分では気づいていない印象を楽しく知る

### 推奨人数

3人から10人程度。

### 所要時間

10分から20分。

### 基本ルール

1. 参加者を登録する
2. 対象者を1人選ぶ
3. 対象者は自分に当てはまる特徴ワードを選ぶ
4. 他の参加者は対象者に当てはまる特徴ワードを選ぶ
5. 結果を4分類で表示する

### 結果分類

- 開放の窓: 自分も他人も選んだ特徴
- 秘密の窓: 自分だけが選んだ特徴
- 盲点の窓: 他人だけが選んだ特徴
- 未知の窓: 誰も選ばなかった特徴

### 機能要件

- 参加者名を登録できる
- 対象者を選択できる
- 特徴ワード一覧を表示できる
- 300語の特徴ワードプールを持つ
- ゲーム開始時に10語、15語、20語、30語、50語、100語、300語から今回使う語数を選べる
- 特徴ワードのカテゴリを選べる
- 自己選択と他者選択を分けて保存できる
- 結果を4象限で表示できる
- 結果画面で選ばれた特徴ワードを一覧できる
- 次の対象者に切り替えられる

### 初期特徴ワード例

- 話しやすい
- 聞き上手
- 行動力がある
- 慎重
- 明るい
- 落ち着いている
- 面倒見がよい
- 発想が豊か
- 冷静
- 盛り上げ上手
- 誠実
- 好奇心が強い
- 頼りになる
- 気配りができる
- 意外性がある

### 注意事項

- 容姿、年齢、恋愛、収入などに踏み込みすぎる特徴ワードは避ける
- 結果を順位化しない
- 「欠点探し」ではなく「印象の共有」として設計する

## 3. ウミガメのスープ

### 概要

参加者が「はい」「いいえ」「関係ありません」で答えられる質問を重ね、謎の真相を推理する水平思考ゲーム。

### 目的

- 会話と推理で場を盛り上げる
- 少人数でも大人数でも参加しやすくする
- 正解までの過程を楽しむ

### 推奨人数

2人から8人程度。

### 所要時間

5分から15分。

### 基本ルール

1. 出題者を決める、またはアプリが問題を表示する
2. 問題文を参加者に見せる
3. 参加者は順番に質問する
4. 出題者は「はい」「いいえ」「関係ありません」で回答する
5. 真相が分かった参加者は回答する
6. 正解後に解説を表示する

### 機能要件

- 300問の問題プールを持つ
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う問題数を選べる
- カテゴリで問題を絞り込める
- 問題文を表示できる
- 出題者向けに答えと解説を表示できる
- 参加者向けには答えを隠せる
- 質問履歴を記録できる
- 質問への回答を「はい」「いいえ」「関係ありません」「補足あり」で記録できる
- ヒントを段階的に表示できる
- 正解後またはギブアップ後に真相を表示できる

### 問題データ要件

- タイトル
- 難易度
- 所要時間目安
- 問題文
- 真相
- 解説
- ヒント
- カテゴリ

### 注意事項

- グロテスク、差別的、過度に暗い問題は避ける
- 飲み会用として、短く解ける問題を優先する
- 初心者向けにヒントを段階的に出せるようにする

## 4. ワードウルフ

### 概要

参加者に似たお題を配り、会話を通じて少数派の参加者を探すゲーム。

### 目的

- 会話を自然に発生させる
- 初対面でも話しやすい空気を作る
- 駆け引きと推理で盛り上げる

### 推奨人数

4人から10人程度。

### 所要時間

5分から10分。

### 基本ルール

1. 参加者を登録する
2. アプリがお題を参加者ごとに割り当てる
3. 多数派と少数派には似たワードが配られる
4. 各自が自分のお題を確認する
5. 制限時間内で会話する
6. 誰が少数派か投票する
7. 結果を表示する

### 機能要件

- 参加者名を登録できる
- お題セットを選択できる
- Hな話題をトグルスイッチでON/OFFできる
- Hな話題OFFでは大人向けカテゴリを表示せず、出題にも含めない
- 少数派人数を設定できる
- 各参加者に個別のお題を表示できる
- 会話タイマーを表示できる
- 投票画面を表示できる
- 投票結果を集計できる
- 勝敗を表示できる

### お題例

- ラーメン / うどん
- 海 / プール
- 映画館 / カラオケ
- コーヒー / 紅茶
- 焼肉 / しゃぶしゃぶ

### 注意事項

- 内輪すぎるお題は初期データに入れない
- センシティブな属性に関するお題は避ける
- 似すぎて会話不能、違いすぎて即バレにならない組み合わせを用意する

## 5. NGワードゲーム

### 概要

参加者ごとに本人には見えないNGワードを設定し、会話中にその言葉を言わせるゲーム。

### 目的

- 普段の会話にゲーム性を足す
- 飲み会中に並行して遊べる
- 短時間で笑いを作る

### 推奨人数

3人から10人程度。

### 所要時間

5分から15分。

### 基本ルール

1. 参加者を登録する
2. 各参加者にNGワードを割り当てる
3. 本人以外にはNGワードを見せる
4. 制限時間内で自由に会話する
5. 自分のNGワードを言った参加者はアウト、または減点
6. 終了時に結果を表示する

### 機能要件

- 参加者名を登録できる
- NGワードを自動割り当てできる
- 手動でNGワードを設定できる
- 本人確認用の個別表示モードを用意する
- タイマーを表示できる
- アウトまたは減点を記録できる
- 結果を一覧表示できる

### NGワード例

- すごい
- なるほど
- たしかに
- やばい
- かわいい
- 忙しい
- 眠い
- おいしい
- 仕事
- 明日

### 注意事項

- 言いにくすぎる単語は避ける
- 個人を傷つける言葉をNGワードにしない
- 脱落制よりも減点制のほうが飲み会では遊びやすい

## 6. 二択トーク

### 概要

参加者が二択のお題に答え、その理由を話すことで会話を広げるゲーム。

### 目的

- 場を温める
- 初対面でも話題を作りやすくする
- 価値観の違いを軽く楽しむ

### 推奨人数

2人から20人程度。

### 所要時間

3分から10分。

### 基本ルール

1. アプリがお題を表示する
2. 参加者はAまたはBを選ぶ
3. 投票結果を表示する
4. 少数派または希望者が理由を話す
5. 次のお題に進む

### 機能要件

- 二択お題をランダム表示できる
- カテゴリを選択できる
- Hな話題をトグルスイッチでON/OFFできる
- Hな話題OFFでは大人向けカテゴリを表示せず、出題にも含めない
- Hな話題ONでは「全部」に大人向けも混ぜ、大人向けカテゴリだけでも遊べる
- A/Bの投票を記録できる
- 投票結果を割合で表示できる
- 次のお題に進める
- お題をスキップできる

### お題例

- 旅行するなら海 / 山
- 朝型 / 夜型
- 甘いもの / しょっぱいもの
- 計画派 / 直感派
- 家でゆっくり / 外に出かける
- 夜のデートならバー / 夜景
- ドキッとするのは手をつなぐ / 目が合う

### 注意事項

- 政治、宗教、収入、恋愛経験など踏み込みすぎるお題は避ける
- 正解を決めず、理由を楽しむ設計にする
- 回答を強制しない

## 7. 第一印象ランキング

### 概要

お題に対して「一番当てはまりそうな人」を投票し、第一印象やイメージを共有するゲーム。

### 目的

- 初対面の会話を作る
- 人の印象をポジティブに共有する
- 場を明るくする

### 推奨人数

4人から12人程度。

### 所要時間

5分から15分。

### 基本ルール

1. 参加者を登録する
2. アプリがお題を表示する
3. 各参加者が該当しそうな人に投票する
4. 結果を表示する
5. 選ばれた人が一言コメントする

### 機能要件

- 参加者名を登録できる
- お題をランダム表示できる
- 通常300問、大人向け30問の設問プールを持てる
- Hな話題をトグルスイッチでON/OFFできる
- ゲーム開始時に今回使う設問数を選べる
- 参加者に投票できる
- 自分への投票をON/OFFできる
- 得票数を表示できる
- 同票の場合は複数人を表示できる
- お題カテゴリを選べる
- 進め方、投票時の注意、結果後の声かけを画面内に表示できる

### お題例

- 一番フットワークが軽そうな人
- 一番聞き上手そうな人
- 一番サプライズが上手そうな人
- 一番おいしいお店を知っていそうな人
- 一番旅行の計画がうまそうな人

### 注意事項

- ネガティブなお題は禁止する
- 容姿やプライベートに踏み込みすぎるお題は避ける
- いじられた人が困らない表現にする

## 8. 匿名質問箱

### 概要

参加者が匿名で質問を投稿し、ランダムまたは順番に表示して回答するゲーム。

### 目的

- 普段聞きにくいが軽く話せる質問を出しやすくする
- 会話のきっかけを作る
- 内輪の場を盛り上げる

### 推奨人数

2人から15人程度。

### 所要時間

5分から20分。

### 基本ルール

1. 参加者が質問を投稿する
2. アプリが質問を一覧またはランダムで表示する
3. 指名された人、または全員が回答する
4. 回答後に次の質問へ進む

### 機能要件

- 匿名質問を投稿できる
- 300問のテンプレート質問プールを持つ
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う質問数を選べる
- 投稿済み質問を一覧できる
- 質問をランダムに表示できる
- 回答済みとして管理できる
- 不適切な質問をスキップできる
- 質問カテゴリを用意できる

### 質問例

- 最近ちょっと嬉しかったことは？
- 子どものころ好きだった遊びは？
- 今年やってみたいことは？
- 旅行で行ってよかった場所は？
- 実は少し得意なことは？

### 注意事項

- 匿名でも相手を困らせる質問は禁止する
- 投稿前に注意文を表示する
- スキップしやすい設計にする

## 9. 定番ゲームパック

### 概要

飲み会で定番の短時間ゲームを、スマートフォン1台で進行できるお題カード集として提供する。

参考にする観点は、準備が少ないこと、短時間で決着すること、誰かを傷つける罰ゲームにしないこと、人数や親密度に合わせて難易度を選べること。

### 目的

- 飲み会の途中で空気を変えたいときにすぐ使える
- ルール説明に時間をかけず、幹事が読み上げるだけで進行できる
- 山手線、逆さ言葉、外来語禁止などの定番遊びを1つの入口にまとめる
- 初対面でも遊びやすいものから、仲の良い場で盛り上がるものまで選べるようにする

### 推奨人数

2人から20人程度。

全部モードは、3人以上を推奨する。

### 所要時間

3分から15分。

### 搭載ゲーム種別

- 山手線: お題に合う言葉をリズムよく順番に言う
- 多数派予想: A/Bのうち、この場で多数派になりそうな方を予想する
- 2真実1嘘: 本当の話2つと嘘1つを出し、聞き手が嘘を当てる
- 逆さ言葉: 出された言葉を逆から読む
- 外来語禁止: カタカナ語を避けてテーマを説明する
- 早打ち: 表示された短文をスマホで早く正確に入力する
- 記憶描き: 見慣れたものを記憶だけで描く
- 価値観メーター: 1から100の強さを数字なしでたとえる
- ひとこと演技: 同じセリフを感情だけで演じ分ける
- ヒント連想: 出題者がヒントを出し、回答者が答えを当てる

### 機能要件

- 参加者名を登録できる
- ゲーム種別を選択できる
- 「全部」モードでは複数種別のお題をランダムに混ぜられる
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べる
- 全体で300問の設問プールを持つ
- 各ゲーム種別ごとに30問以上を持つ
- 各お題カードに、進め方、判定方法、注意点を表示する
- 答えを隠す必要があるゲームでは、答え表示を任意に切り替えられる
- 全問終了後に、もう一度遊ぶか設定へ戻れる

### 進行要件

- 勝敗は厳密にしすぎず、会話が続くことを優先する
- 1問ごとに司会または親を表示する
- テンポが落ちたら、司会判断で次のお題へ進める
- 外部サービスや専用道具が必要なゲーム名は、そのまま再現せず、アプリ内で遊べる汎用ルールに置き換える

### 注意事項

- 特定の参加者を貶める罰ゲームにしない
