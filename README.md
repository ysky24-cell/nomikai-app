# 飲み会用アプリ

飲み会・懇親会・チーム交流で使えるミニゲーム集アプリです。

トップ画面を各ゲームへの入口にし、参加者がその場でスマートフォンからすぐ遊べる体験を目指します。

## 初期搭載候補

- ジョハリの窓
- ウミガメのスープ
- ワードウルフ
- NGワードゲーム
- 二択トーク
- 第一印象ランキング
- 匿名質問箱
