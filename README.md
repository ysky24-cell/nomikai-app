# 飲み会用アプリ

飲み会・懇親会・チーム交流で使えるミニゲーム集アプリです。

トップ画面を各ゲームへの入口にし、参加者がその場でスマートフォンからすぐ遊べる体験を目指します。

## 搭載ゲーム

- 山手線ゲーム
- 二択トーク
- ワードウルフ
- NGワードゲーム
- 第一印象ランキング
- 定番ゲームパック
- ジョハリの窓
- ウミガメのスープ
- 匿名質問箱

## URL候補ゲーム

スペースマーケットマガジンの飲み会ゲーム記事を参考に、URL候補をトップ画面から個別に遊べる入口として追加しています。

- 既存の山手線ゲーム、第一印象ランキングに加え、マジョリティゲーム、2つの真実と1つの嘘、カウントアップゲーム、逆さ言葉ゲーム、曲名連想クイズ、お絵描きクイズ、ドキドキはずれカード、スマホ早打ちゲーム、記憶だけでロゴを書く、価値観メーター、ひとこと演技ゲーム、飲み会すごろく、シンプル陣取り、変な一言カルタ、エモヒント連想、資源交渉トーク、人生イベントすごろく、腕相撲トーナメント、安全はずれ抽選、人物当てヒントクイズ、大人数マジョリティ、鼻歌イントロドン、外来語禁止ゲームを個別入口化しています
- 外部アプリや市販ゲームは完全再現せず、静的ページで遊べる汎用版として再構成しています
- 追加したURL候補ゲームは、それぞれ通常300問、大人向け30問のプールを持ちます
- 全URL候補ゲームでHな話題をトグルスイッチでON/OFFできます
- トップ画面では「全部」「URL候補」「会話」「反射」「運」「描く」「ボード風」「大人数」で絞り込めます

## データ保存

初期版ではデータベースを使いません。参加者、投票、ゲーム進行はブラウザ内の状態と`localStorage`に保存します。

トップ画面とゲーム中の画面から全体を初期化できます。初期化すると、全ゲームの参加者、投票、進行状況を消してトップ画面へ戻ります。

## 山手線ゲーム

- 通常300問、大人向け30問のお題プールを搭載しています
- スペースマーケットマガジンの飲み会ゲーム記事にある「今年流・山手線ゲーム」を、静的アプリ内で進行できる専用ゲームとして実装しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使うお題数を選べます
- Hな話題はトグルスイッチでON/OFFできます
- カテゴリは「全部」「定番」「食べ物」「飲み会」「仕事仲間」「日常」「エンタメ」「おでかけ」「季節」「思い出」「もしも」「大人向け」から選べます
- 1人の持ち時間は3秒、5秒、10秒から選べます
- 出た答え、パス、アウト回数を画面内で記録できます

## 二択トーク

- 通常300問、大人向け30問の設問プールを搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- Hな話題はトグルスイッチでON/OFFできます
- OFFでは「大人向け」カテゴリを表示せず、出題にも含めません
- ONでは「全部」に大人向けも混ざり、「大人向け」カテゴリだけでも遊べます
- カテゴリは「全部」「アイスブレイク」「食べ物」「休日」「暮らし」「会話」「エンタメ」「おでかけ」「仕事仲間」「もしも」「日常」から選べます
- 進め方、投票時の注意、結果後の声かけを画面内に表示します

## ワードウルフ

- 300ペアのお題プールを搭載しています
- 進行役が迷わないよう、ゲーム開始前に進め方、会話ルール、投票ルールを表示します
- Hな話題はトグルスイッチでON/OFFできます
- OFFでは「大人向け」カテゴリを表示せず、出題にも含めません
- ONでは「全部」に大人向けも混ざり、「大人向け」カテゴリだけでも遊べます
- カテゴリは「全部」「食べ物」「日常」「おでかけ」「エンタメ」「旅行」「仕事仲間」「人柄」「恋バナ」「夜の話題」から選べます

## NGワードゲーム

- 参加者ごとに本人だけ見ないNGワードを配ります
- 進め方、配布時の注意、会話中の記録方法、同点時の扱いを画面内に表示します
- 初期版は脱落なしの減点制です

## 第一印象ランキング

- 通常300問、大人向け30問の設問プールを搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- Hな話題はトグルスイッチでON/OFFできます
- カテゴリは「全部」「はじめやすい」「人柄」「会話」「飲み会」「仕事仲間」「友だち」「暮らし」「遊び」「おでかけ」「もしも」「大人向け」から選べます
- 自分への投票はトグルスイッチでON/OFFできます
- 選ばれた人が嬉しくなるよう、進め方、投票時の注意、結果後の声かけを画面内に表示します

## 定番ゲームパック

- 300問の設問プールを搭載しています
- スペースマーケットマガジンの飲み会ゲーム記事を参考に、短時間で決着し、誰かを傷つけにくいゲームとして再構成しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- 種類は「全部」「山手線」「多数派予想」「2真実1嘘」「逆さ言葉」「外来語禁止」「早打ち」「記憶描き」「価値観メーター」「ひとこと演技」「ヒント連想」から選べます
- 各ゲームごとに、進め方、判定方法、盛り上げ方の注意を画面内に詳しく表示します
- 答えを隠す必要がある「逆さ言葉」「ヒント連想」は、出題者だけが答えを確認できる表示にしています

## ジョハリの窓

- 300語の特徴ワードを搭載しています
- ゲーム開始時に10語、15語、20語、30語、50語、100語、300語から今回使う語数を選べます
- カテゴリは「全部」「温かさ」「会話」「安定感」「発想」「リード」「支える力」「仕事仲間」「伸びしろ」「遊び心」「個性」から選べます
- 本人の選択、他者の選択を分けて進め、結果を「開放」「盲点」「秘密」「未知」の4つの窓で表示します
- 欠点探しにならないよう、結果後の話し方も画面内に表示します

## ウミガメのスープ

- 300問の短問プールを搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う問題数を選べます
- カテゴリは「全部」「日常」「食べ物」「仕事仲間」「おでかけ」「イベント」「学校」「家」「お店」「連絡」「外」から選べます
- 出題者だけが真相を表示でき、ヒントは段階的に開けます
- 質問の返し方、詰まった時の進め方を画面内に表示します

## 匿名質問箱

- 300問のテンプレート質問を搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う質問数を選べます
- カテゴリは「全部」「軽め」「日常」「食べ物」「思い出」「人間関係」「仕事仲間」「価値観」「遊び」「おでかけ」「これから」から選べます
- その場で匿名質問を追加し、テンプレート質問と混ぜてランダムに引けます
- 投稿者探しや答えにくい質問を避ける注意文、スキップ導線を画面内に表示します
