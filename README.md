# 飲み会用アプリ

飲み会・懇親会・チーム交流で使えるミニゲーム集アプリです。

トップ画面を各ゲームへの入口にし、参加者がその場でスマートフォンからすぐ遊べる体験を目指します。

## 初期搭載候補

- 二択トーク
- ワードウルフ
- NGワードゲーム
- 第一印象ランキング
- 定番ゲームパック

以下は追加予定です。

- ジョハリの窓
- ウミガメのスープ
- 匿名質問箱

## データ保存

初期版ではデータベースを使いません。参加者、投票、ゲーム進行はブラウザ内の状態と`localStorage`に保存します。

トップ画面とゲーム中の画面から全体を初期化できます。初期化すると、全ゲームの参加者、投票、進行状況を消してトップ画面へ戻ります。

## 二択トーク

- 通常300問、大人向け30問の設問プールを搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- Hな話題はトグルスイッチでON/OFFできます
- OFFでは「大人向け」カテゴリを表示せず、出題にも含めません
- ONでは「全部」に大人向けも混ざり、「大人向け」カテゴリだけでも遊べます
- カテゴリは「全部」「アイスブレイク」「食べ物」「休日」「暮らし」「会話」「エンタメ」「おでかけ」「仕事仲間」「もしも」「日常」から選べます
- 進め方、投票時の注意、結果後の声かけを画面内に表示します

## ワードウルフ

- 300ペアのお題プールを搭載しています
- 進行役が迷わないよう、ゲーム開始前に進め方、会話ルール、投票ルールを表示します
- Hな話題はトグルスイッチでON/OFFできます
- OFFでは「大人向け」カテゴリを表示せず、出題にも含めません
- ONでは「全部」に大人向けも混ざり、「大人向け」カテゴリだけでも遊べます
- カテゴリは「全部」「食べ物」「日常」「おでかけ」「エンタメ」「旅行」「仕事仲間」「人柄」「恋バナ」「夜の話題」から選べます

## NGワードゲーム

- 参加者ごとに本人だけ見ないNGワードを配ります
- 進め方、配布時の注意、会話中の記録方法、同点時の扱いを画面内に表示します
- 初期版は脱落なしの減点制です

## 第一印象ランキング

- 300問の設問プールを搭載しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- カテゴリは「全部」「はじめやすい」「人柄」「会話」「飲み会」「仕事仲間」「友だち」「暮らし」「遊び」「おでかけ」「もしも」から選べます
- 自分への投票はトグルスイッチでON/OFFできます
- 選ばれた人が嬉しくなるよう、進め方、投票時の注意、結果後の声かけを画面内に表示します

## 定番ゲームパック

- 300問の設問プールを搭載しています
- スペースマーケットマガジンの飲み会ゲーム記事を参考に、短時間で決着し、誰かを傷つけにくいゲームとして再構成しています
- ゲーム開始時に5問、10問、20問、30問、50問、100問、300問から今回使う設問数を選べます
- 種類は「全部」「山手線」「多数派予想」「2真実1嘘」「逆さ言葉」「外来語禁止」「早打ち」「記憶描き」「価値観メーター」「ひとこと演技」「ヒント連想」から選べます
- 各ゲームごとに、進め方、判定方法、盛り上げ方の注意を画面内に詳しく表示します
- 答えを隠す必要がある「逆さ言葉」「ヒント連想」は、出題者だけが答えを確認できる表示にしています
